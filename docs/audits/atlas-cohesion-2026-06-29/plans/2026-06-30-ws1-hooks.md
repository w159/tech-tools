# Atlas WS1 — Hook De-misfiring + Orchestration Marker (Phase 0 + WS1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stop the atlas dispatch tripwire, completion gate, and self-improvement nudge from firing on ordinary non-orchestration sessions, by introducing an explicit per-session orchestration marker that only real atlas orchestration sets.

**Architecture:** Add an `orchestrating` flag to the `runs` table (DB-authoritative, keyed per session, so no cross-session staleness) plus a human-visible sentinel file. The three Stop/PostToolUse hooks gate their nag/block behavior on `atlas_db.is_orchestrating(session_id)`. The flag is set by `atlas_db.mark_orchestrating(...)`, which the `atlas-engine` skill (and later `atlas-launch`) calls when orchestration actually begins. Telemetry logging is unchanged; only the user-facing nag/block is gated.

**Tech Stack:** Python 3 stdlib (`sqlite3`, `unittest`), Claude Code hooks (JSON payload on stdin), the existing `plugins/atlas/scripts/atlas_db.py` observability layer.

## Global Constraints

- Prose/comments: U.S.-keyboard ASCII only. No em dashes, en dashes, curly quotes, or ellipsis characters. Use `-`, `,`, `:`, `()`, or split sentences.
- Hooks must stay fail-open: any exception exits 0 and never blocks or wedges a session. Preserve the existing `try/except: pass` and `sys.exit(0)` discipline in every hook.
- DB changes must be idempotent on pre-existing databases: follow the `ALTER TABLE ... ADD COLUMN` + `except sqlite3.OperationalError: pass` pattern already in `init()`.
- No `npm install` / dependency work under the iCloud repo path. These are Python stdlib changes; run tests in place with `python3 -m unittest`.
- Tests mirror the existing harness exactly: subprocess + stdin JSON + `ATLAS_DB` tmpdir for end-to-end hook tests (tripwire style); direct import + call for pure-helper tests (completion_gate style).
- Propagation (repo CLAUDE.md): a user-visible change lands across all owning layers. WS1 touches hooks + `atlas_db.py` + the four hook-inventory doc surfaces; keep them consistent in Task 6.
- Verification protocol: every task ends by running its test command and observing real output before checking the box. No "should pass."

**Marker contract (used by every task below):**
- `atlas_db.mark_orchestrating(conn, session_id, cwd=None) -> int` — sets `runs.orchestrating=1` for this session's current-or-latest run (creates a run if none), optionally writes the sentinel, returns the run_id.
- `atlas_db.is_orchestrating(conn, session_id) -> bool` — True iff this session's current-or-latest run has `orchestrating=1`.
- Sentinel file (advisory only, never read for gating): `<cwd>/docs/.run/atlas-engine.active`.

```python
def mark_orchestrating(conn, session_id, cwd=None):
    """Flag this session's run as a real atlas orchestration run. Idempotent.
    Creates a run if none exists yet (e.g. the boot hook has not fired).
    Optionally writes an advisory sentinel under <cwd>/docs/.run/."""
    rid = current_run_id(conn, session_id) or latest_run_id(conn, session_id)
    if rid is None:
        base = cwd or "."
        pid = register_project(conn, base, os.path.basename(os.path.abspath(base)))
        rid = start_run(conn, pid, session_id)
    conn.execute("UPDATE runs SET orchestrating=1 WHERE id=?", (rid,))
    conn.commit()
    if cwd:
        _write_orchestration_sentinel(cwd)
    return rid


def is_orchestrating(conn, session_id):
    """True when this session's current-or-latest run is flagged orchestrating."""
    rid = current_run_id(conn, session_id) or latest_run_id(conn, session_id)
    if rid is None:
        return False
    row = conn.execute(
        "SELECT orchestrating FROM runs WHERE id=?", (rid,)
    ).fetchone()
    return bool(row and row[0])


def _write_orchestration_sentinel(cwd):
    """Advisory only. Never read for gating; a stale file must not enable a gate."""
    try:
        run_dir = os.path.join(cwd, "docs", ".run")
        os.makedirs(run_dir, exist_ok=True)
        with open(os.path.join(run_dir, "atlas-engine.active"), "w") as f:
            f.write(str(time.time()))
    except Exception:
        pass  # sentinel is best-effort
```
